Detalles de la aplicacion

La aplicacion tiene como funcion central la busqueda de restaurantes por Estados Unidos.

Esta tambien provee de Busquedas para saber que ciudades tenemos disponibles para buscar y asi como tambien permite ver que tipo de comida sirven en ciertos lugares.

Cuando se buscan restaurantes por ciudad y no tienen foto es porque la API para algunas fotos requiere un key socio con el cual no cuento.

Actualmente se encuentra optimizada para dispositivos moviles, para dispositivos mas grandes puede llegar a haber fixes futuros.
